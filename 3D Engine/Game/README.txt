## Controls and handling instructions

- Camera control:

Press F to focus on the model loaded and visible on screen.
ALT + Left Mouse Button will allow you to orbitate around the model loaded and visible on screen.
Right Mouse Button activates WASD controls:
	W - Forward
	S - Backward
	A - Left
	D - Right
 	Left Shift - Increments the movement speed
	Space Bar - Increments the height
 	Left Control - Decrements the height
Use the mouse wheel to zoom in and out.

The models and textures should go inside their specified folders, despite not being
necessary, it maintains the files organized.

On the main menu bar:
	Press File to present the Exit button, which once pressed will end the application.
	Press Window to present several options:
		Grid contains functions to show or hide the grid and the axis.
		Configuration shows the configuration window.
		Console shows the console window.
		Inspector shows the inspector window.
	Press About to find out more about the creators and the engine itself.
